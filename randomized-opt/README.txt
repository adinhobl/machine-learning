To run the code for this project follow these steps:

1. If you don't have Anaconda installed, go to the following link and install 
    MiniConda for your platform:
    https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html 
2. Git clone the project and cd into the randomized-opt directory:
    https://github.com/adinhobl/machine-learning.git 
3. Activate the conda environment uing the commands found here:
    https://docs.conda.io/projects/conda/en/latest/user-guide/tasks/manage-environments.html?highlight=environment.yml#creating-an-environment-from-an-environment-yml-file 
4. To run the code from the terminal, type in `python {file}.py` where {file} is replaced by
    whichever file you want to run: nn, opt1, opt2, opt3. This will generate the figures and 
    terminal printout.
